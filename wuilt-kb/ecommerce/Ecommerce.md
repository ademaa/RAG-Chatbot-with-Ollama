## How to Set Up Facebook Conversion API for Your Wuilt Store
**This guide will walk you through setting up the Facebook Conversion API for your Wuilt store. **

*Category: التطبيقات*

What Is the Facebook Conversion API?


The Facebook Conversion API allows you to send server-side events directly to Facebook, ensuring your data remains accurate, even with browser restrictions like cookie blocking. Wuilt has simplified the integration process for you, so once you set up your Pixel or Conversion API access token, we’ll handle everything in the background.




Steps to Install Facebook Conversion API


If you’ve already installed a Facebook Pixel on your website, follow these steps to set up the Conversion API.




Step 1: Get Your Dataset ID




In 
Facebook Events Manager
, locate the 
Data Sources
 section.


Select your Pixel and note down your 
Dataset ID
 under the settings.








Step 2: Get Your Access Token




Go to Ads Manager > Left Tab > All Tools > Events Manager


In the events manager, select Data Sources to view all your created pixels.


Select your pixel and go to Settings


Scroll down till you reach Conversions API > Setup Manually


Click on Generate access token






Step 3: Add the Access Token and Dataset ID to Wuilt




Log in to your 
Wuilt Dashboard
.


Go to 
App Center  > Scroll down to Meta Pixel and Conversion API and click on connect






3. 
Enter your Access Token and Dataset ID in the respective fields.


4. Click 
Connect
 to complete the integration.




This Is an Interactive Guide


This interactive guide is designed to simplify the process for you. It provides clear, step-by-step instructions to help you set up Facebook Conversion API for your Wuilt store

When adding a new product, you'll need to provide the following information:




Product Name:
 A clear and concise name that accurately describes the product.


Detailed Description:
 A comprehensive description that highlights the product's features, benefits, and specifications. This helps customers understand what the product is and why they need it.


Price:
 The selling price of the product.


Images:
 High-quality images of the product from multiple angles. Good visuals can significantly impact a customer's purchasing decision.


Category:
 The category or department that the product belongs to (e.g., clothing, electronics, home goods). This helps customers navigate your store more easily.


Inventory:
 The quantity of the product that is currently available for sale. This helps prevent overselling.

 The weight and dimensions of the product, which are necessary for calculating shipping costs.
